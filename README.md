Program 1 : convertion.c

Description :
This program converts a given number of days into years, months, and remaining days.

How it works :
Takes input in total days
Converts:
1 year = 365 days
1 month = 30 days
Outputs the equivalent years, months, and days

Example :
Input: 400  
Output:  
Year : 1  
Month : 1  
Days : 5  


Program 2 : check_leep_year.c

Description:
This program checks whether a given year is a leap year or not.

Logic Used:
A year is a leap year if:
It is divisible by 400, OR
It is divisible by 4 but not divisible by 100

Example :
Input: 2024  
Output: 2024 is a Leap Year.
